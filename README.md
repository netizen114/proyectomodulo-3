# Alke Wallet

## Descripción

Proyecto final del Módulo 3, consistente en la creación y manipulación de una base de datos utilizando SQL.

El repositorio contiene los siguientes archivos:

- README.md
- Proyecto final modulo 3.sql: contiene las sentencias SQL en orden para la creación de la base de datos y las operaciones posteriores sobre ella (DDL y DML).
- documentacion.pdf: documento con las sentencias SQL y capturas de pantalla que evidencian la creación y manipulación de la base de datos.

## Tecnologías utilizadas

- SQL
- MySQL

## Objetivo del proyecto

Aplicar los conocimientos de modelado de bases de datos, creación de tablas, relaciones mediante claves foráneas y ejecución de operaciones DML.
